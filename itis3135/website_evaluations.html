Website Evaluations

This page provides a critical review of two websites using the CRAP design principles.

Apple

  • Contrast: Excellent. Apple uses high contrast between text and background, typically black on white or white on black, making content very legible. Call-to-action buttons ("Buy", "Learn more") stand out clearly against the clean background.
  • Repetition: Strong. Consistent typography (San Francisco font), iconography, and layout patterns are used throughout the site, reinforcing the brand identity and making navigation predictable.
  • Alignment: Precise. Elements are strictly aligned to a grid system. Text, images, and other components are intentionally placed, creating a clean and organized look that guides the eye naturally down the page.
  • Proximity: Effective. Related information is grouped together (e.g., product image with its name and price), while white space is used generously to separate distinct sections, reducing cognitive load.

Berkshire Hathaway

  • Contrast: Poor. The site uses default blue hyperlinks on a white background with black text. While legible, it lacks visual hierarchy or distinct contrast for important elements vs. less important ones. It feels outdated.
  • Repetition: Weak. There is little stylistic repetition other than the default HTML styling. There is no consistent branding, logo usage, or modern UI components repeated across the site to create a cohesive experience.
  • Alignment: Basic. The layout seems to rely on default browser alignment (left-aligned lists and centered text). It lacks a sophisticated grid or intentional alignment strategy found in modern web design.
  • Proximity: Mixed. Links are listed in a somewhat grouped manner, but without visual separation or grouping containers, the page feels like a raw list of files rather than structured content.
